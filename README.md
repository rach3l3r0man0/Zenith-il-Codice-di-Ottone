# Zenith: il Codice di Ottone

Final project for the **Web Design** course - BSc in Computer Engineering.

## Project Description
This is a full-stack web application developed using HTML, CSS, JavaScript, PHP, and MySQL. 
*Note: The project is currently available in Italian. An English version will be released in the future.*

## How to run the project
To run this project locally, follow these steps:

1. **Install a local server**: Ensure you have a local environment like [XAMPP](https://www.apachefriends.org/).
2. **Clone the repository**.
3. **Setup the Database**:
   * Open phpMyAdmin.
   * Create a new database named [romano_636006].
   * Import the .sql file found in the project folder (usually in /database or /assets).
4. **Configure PHP**:
    * Move the project folder to the htdocs directory (XAMPP).
    * Update the database connection credentials in your PHP config file (host, username, password).
5. **Launch**:
   * Open your browser and go to `http://localhost/zenith-il-codice-di-ottone`.
